# BigToken - BOT
Fungsi script ini untuk registrasi dan verifikasi otomatis.
Dibuat dengan NodeJS.
 
# Bot ini sudah tidak aktif lagi. Saya tidak akan mengubahnya.
 
# Requirement
- nodejs 8.6.0 +
# How to use
```sh
$ git clone https://github.com/siikuder/bigtoken-bot.git
$ cd bigtoken
$ npm install
$ node index.js
```
